# AlgoVision

**AlgoVision** is an interactive JavaFX application for visualizing algorithms in real time.  

---

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Ork2004/AlgoVision.git
   cd AlgoVision
   ```
2. Build and run with Maven:
   ```bash
   mvn clean javafx:run
   ```
3. Or run `Main.java` directly in IntelliJ IDEA.

---

## Planned Features

- Merge Sort, Quick Sort, Heap Sort
- Graph algorithms (BFS, Dijkstra, Kruskal)
- Pathfinding visualization (A*, Maze Solver)
- Customizable color themes
- Interactive element highlighting
